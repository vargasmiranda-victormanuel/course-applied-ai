## Principales Tendencias y Tecnologías
- **MCP (Model Context Protocol):**
  - Desarrollado por Anthropic.
  - Permite conectar LLMs con herramientas y fuentes de datos externas.
  - Emergiendo como estándar de la industria.

- **OpenAI Agents SDK:**
  - Framework oficial de OpenAI.
  - Soporta handoffs (transferencias), guardrails (marcos de seguridad) y trazabilidad nativa.

- **CrewAI:**
  - Framework para agentes multi-funcionales.
  - Coordina agentes especializados colaborando como un equipo con roles y tareas definidos.

- **Computer Use por Claude (Anthropic):**
  - Capacidad para controlar interfaces gráficas de usuario simulando comportamiento humano.

- **Agentic Workflows:**
  - Implementaciones de LLMs para tomar decisiones y acciones autónomas en múltiples pasos.
